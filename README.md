# Social Media Clone (Laravel + Tailwind CSS)

A simple social media application where users can:
- Register and login
- Create posts
- Delete their own posts
- View a dashboard

Built with **Laravel**, **Blade templates**, **Tailwind CSS**, and **SQLite/MySQL**.
